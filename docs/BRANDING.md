# Agent Messaging Protocol - Branding Guidelines

## Logo

**Primary Logo:** `{AMP}` - JSON-style typography representing the protocol's data format

**Logo Files:**
- `assets/images/amp-logo-trimmed.png` - Primary logo (628x304px, transparent background)
- `assets/images/amp-logo.png` - Original with padding
- `assets/images/amp-logo-transparent.png` - Transparent version with padding

## Logo Sizing

| Context | Height | Notes |
|---------|--------|-------|
| Website header | 24px | Standard navigation size |
| Website footer | 24px | Matches header |
| Favicon | 16px, 32px | Multi-size ICO |
| Apple Touch Icon | 180px | iOS home screen |
| Android Chrome | 192px, 512px | PWA icons |
| Open Graph | 1200x630 | Social sharing |
| Twitter Card | 1200x600 | Twitter sharing |

## Colors

**Primary Palette:**
- Slate 900: `#1e293b` - Primary text, dark backgrounds
- Slate 600: `#475569` - Secondary text
- Slate 50: `#f8fafc` - Light backgrounds

**Brand Colors:**
- The `{AMP}` logo uses dark slate tones for professional appearance

## Typography

**Primary Font:** Space Grotesk (headings, body)
**Monospace Font:** JetBrains Mono (code, logo reference)

## Usage Guidelines

1. Always use the trimmed logo file for display
2. Maintain 24px height for navigation contexts
3. Logo should have transparent background for flexibility
4. Minimum clear space: 8px around logo
